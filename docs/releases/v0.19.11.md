# Version v0.19.11

This release includes important bug fixes to improve the usability and compatibility of the llamabot CLI, particularly addressing issues with optional dependencies.

## New Features

- None in this release.

## Bug Fixes

- Fixed an issue where the llamabot CLI failed to load without uvicorn installed. This update ensures uvicorn is only imported when necessary, preventing `ModuleNotFoundError` during CLI operations. (835803f) (Eric Ma)

## Deprecations

- None in this release.
