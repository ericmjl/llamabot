# Version v0.19.0

This release introduces significant enhancements to the MCP (Multi-Call Platform) tools, including new features for better integration and usability, alongside updates to testing frameworks and documentation. Additionally, there are updates to dependencies and internal model lists.

## New Features

- Introduced first-class support for MCP tools and a new marimo-pair skill, enhancing the MCPClientManager and session persistence. (114862) (Eric Ma)
- Added support for FastMCP shorthand in MCP server configurations, allowing simplified API usage. (02370d) (Eric Ma)
- Updated MCP tutorial notebook to align with Anthropic-only model options and integrated a new AgentBot test for multi-tool selection. (90f8cf) (Eric Ma)
- Added a new MCP tools AgentBot marimo tutorial in the documentation, demonstrating the integration of FastMCP tools with local Python tools. (b76844) (Eric Ma)
- Refactored MCP server configuration model for clearer API terminology. (03072f) (Eric Ma)

## Bug Fixes

- Fixed an issue to preserve tool-call context across decision loops in AgentBot, preventing repeat-loop behavior. (ce4e1c) (Eric Ma)

## Deprecations

- No deprecations were introduced in this version.
