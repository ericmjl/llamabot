# Version v0.19.8

This release includes significant improvements to the visualization of waterfall spans and enhancements in the SimpleBot span instrumentation. Additionally, there are updates to the notebook environment, improving clarity and functionality.

## New Features

- Enhanced waterfall span visualization for better alignment and clarity. This update includes changes such as the adjustment of nesting indents, removal of unnecessary background space, addition of an elapsed-time axis, and improved multi-epoch positioning. (01a3b65) (Eric Ma)
- Updated SimpleBot span instrumentation to include new child spans for message composition, token generation, and response processing, providing clearer separation of tasks and timings. (01a3b65) (Eric Ma)
- Improved notebook usability by naming all cells descriptively, removing private variables, and updating the model to a newer version. New demo cells for waterfall and multi-turn interactions have also been added. (01a3b65) (Eric Ma)

## Bug Fixes

- Fixed accuracy issues in waterfall span visualization by adjusting the alignment and scaling of elements. (01a3b65) (Eric Ma)

## Deprecations

- There are no deprecations in this version.
