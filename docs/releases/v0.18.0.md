# Version v0.18.0

This release introduces significant enhancements to the Async* bots, including new demos and design documentation updates. It also includes improvements to the CI workflows and updates to dependencies.

## New Features

- Introduced AsyncAgentBot with native async decide path and comprehensive design documentation updates. (366e1ad) (Eric Ma)
- Added HTMX + SSE demos for AgentBot and AsyncToolBot, showcasing real-time streaming capabilities. (c9140fd) (Eric Ma)
- Implemented AsyncQueryBot HTMX + SSE demo with a BM25 corpus, enhancing interactive query capabilities. (bf8c1a5) (Eric Ma)
- Added AsyncStructuredBot HTMX + SSE form-filling demo, enabling real-time data extraction and display. (a1f817b) (Claude)
- Introduced initial skill documents for the marimo notebook ecosystem, expanding the toolkit for developers. (198d6e4) (Eric Ma)
- Added SSE streaming utilities and examples, providing tools for structured event streaming from LlamaBot integrations. (db26b76) (Eric Ma)

## Bug Fixes

- Fixed CI workflow to run llamabot via pixi, addressing path issues on GitHub runners. (a79e9f0) (Eric Ma)
- Corrected default settings for structured demo in AsyncStructuredBot, improving initialization and user preferences. (d32d766) (Eric Ma)

## Deprecations

- Removed async_bots module, consolidating Async* bot classes with their synchronous counterparts for better maintainability. (c3ec52c) (Eric Ma)
- Deprecated ToolBot.acall in favor of AsyncToolBot.__call__, streamlining the API for decision paths. (58e7b99) (Eric Ma)
