# Version v0.19.9

This release includes important bug fixes related to logging behavior in the llamabot module, ensuring better control over log output and preserving user-configured log sinks. Additionally, documentation has been improved to clarify the behavior of logging functions.

## New Features

- Added detailed inline documentation for the logger removal operation to clarify its impact on default and user-added sinks. (d51d073) (copilot-swe-agent[bot])

## Bug Fixes

- Modified the llamabot logging setup to only remove the Loguru default sink, preventing the removal of user-added sinks. This change includes regression tests to ensure the correct behavior. (e16b5a5) (Eric Ma)
- Fixed an issue where the llamabot module would not correctly remove the default Loguru DEBUG sink before adding its own, causing unwanted debug output regardless of the configured log level. (7d56b44) (Eric Ma)

## Deprecations

- No deprecations in this release.
