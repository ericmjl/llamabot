# Version v0.19.7

This release introduces significant enhancements to the AgentBot's performance monitoring capabilities, including new spans for better visibility and a new waterfall chart view for span timelines.

## New Features

- Enhanced AgentBot monitoring with new spans for critical operations, ensuring all major time gaps are now visible. This update includes spans for decision-making processes, tool selection, and HTTP requests, providing a comprehensive view of execution times. Additionally, a new waterfall chart view has been added to improve the visualization of span timelines, making it easier to analyze the performance of different components. (f829f39) (Eric Ma)
