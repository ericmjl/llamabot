# Version v0.19.4

This release includes critical bug fixes and improvements in the CI configurations to enhance the development workflow and ensure the integrity of the package without unnecessary dependencies.

## New Features

No new features were added in this version.

## Bug Fixes

- Improved CI checks to only consider the subject line for determining version bump commits, reducing false positives. (afb2361) (Eric Ma)
- Updated the CI process for bare-package tests to ensure that the package installs correctly without any extras, and fixed issues related to release workflows. This includes adjustments to the testing scripts and the release process to prevent errors related to double-triggering and detached-HEAD pushes. (7864f68) (Eric Ma)
- Modified import statements in the codebase to lazily import `numpy` only in methods that require it, avoiding unnecessary core dependency on `numpy`. (e11bef1) (Eric Ma)

## Deprecations

No deprecations were introduced in this version.
