# CLI and examples

LlamaBot ships with a Typer-based CLI under the ``llamabot`` console script.
Commands cover documentation, notebooks, repository helpers, and more.

Example scripts and Marimo notebooks live under ``docs/examples/``. HTMX and
FastAPI demos show how to combine ``AsyncSimpleBot``, ``AsyncStructuredBot``,
or ``AsyncQueryBot`` with streaming responses in the browser.
