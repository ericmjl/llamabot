AgentBot demo

AgentBot runs a PocketFlow graph (decide → tools → …). The server is not token-streaming the LLM inside the graph; you get a status line, then the final result and a memory trace after the run. Demo tool: inch_to_cm. Try: “Convert 12 inches to cm” or “What’s the date today?”

Run: pixi run uvicorn docs.examples.async_agentbot_htmx_demo:app --reload
